# DDUP1.0 Skill

> 多智能体协作平台 - 用自然语言快速创建AI团队

## 触发关键词

- "创建团队"
- "多Agent协作"
- "DDUP"
- "一个指令一个团队"

## 功能说明

### 1. 团队创建

用户告诉DDUP想要什么团队，DDUP自动创建多Agent协作团队。

**示例**：
- "帮我创建一个短视频团队"
- "创建一个数据分析团队"
- "做一个产品发布团队"

### 2. 预置模板

| 模板 | Agent数量 | 角色 |
|------|----------|------|
| 短视频团队 | 3 | 编导、拍摄、剪辑 |
| 数据分析团队 | 3 | 数据收集、分析、报告 |
| 产品发布团队 | 3 | 市场调研、文案、运营 |

### 3. 任务执行

1. 用户输入需求
2. DDUP理解意图
3. 创建对应Agent团队
4. 分发任务给各Agent
5. 汇总结果输出

## 使用方法

```
@DDUP 帮我创建一个短视频团队
```

## 技术实现

- 主Agent：负责意图理解、任务分发、结果汇总
- 子Agent：通过sessions_spawn创建
- 记忆存储：共享memory目录
